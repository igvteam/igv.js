GTEX EQTL example

This example used the special GTEX search mode (enabled with the enableQTLSearch property). Searching for a gene or snp from the search box highlights the associated features. In the case of a snp this includes all eqtls for the snp, and associated genes. For a gene search this includes the gene and its associated eqtls.

Try searching for the snp rs11704810 or the gene SMARCB1
