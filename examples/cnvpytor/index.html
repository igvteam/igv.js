cnvpytor Tracks

  • cnvpytorTrack.html Reads a cnvpytor file and generate view
  • cnvpytorTrackVCF.html Read and process a large VCF file (~3,000,000 sites) for read depth and BAF information, and creates a view. User need to wait a bit as it process the input file on fly
  • cnvpytorTrackVCF_SMALL.html Read and process a small VCF file (~16,500 sites) for read depth and BAF information, and creates a view. User need to wait a bit as it process the input file on fly
